# 角色设计文档

## 角色列表概览

角色列表概览
主要角色（有羁绊剧情的角色）：

1. **王行知/三丰（主角 / 玩家 POV）** - 故事主角，不定积分乐队的吉他手，兼调音负责人。性格温和，但执念深、孤独的燃烧者。故事的理性与疯狂交汇点。

2. **包晏澜（包姐）** - 三丰在不定积分乐队的学姐主唱，是主角的精神引路人。象征"理想之光"的残影，退场后成为主角的执念与幻影。

3. **沈澜 / 林梓 / 林知夏（kiki）** - 贝斯手，唯一救赎。理性温柔的支撑者，经历离开与回归，代表"恒温的理解"。

4. **姜芮萌（辰澈）** - 后辈吉他手。技术很强的后辈吉他手，家境优渥，但为人处世的能力令人堪忧。搞不清乐队矛盾的爆发原因，最后同主角一同迷失在伤痛之中。

5. **方芮宁（方顾）** - 后辈女主唱，看似温柔可爱，实际内心冷漠，是虚假的光。聪明现实的掌控者，以温柔的表象压垮理想。

6. **赵启扬（翔太郎）** - 鼓手，友情的背叛者。热血外向却功利，最痛的背叛者，象征"信任的坍塌"。

7. **江昱 / 沈砺（梓兮）** - 镜像角色。主角的黑面反射，竞争与嫉妒并存。象征"光的阴影"。

次要人物：

1. **马浩宇（对手乐队队长 / 镜像投射）** - 自信功利型，对主角构成"如果理想顺利实现，他可能成为的样子"之反射。

2. **焱野（对手乐队键盘手）** - 朋克姐，技术强，对音乐抱有热情，只和她认可的强者一起玩乐队，小白cp，追小白的事情是一段趣事。

3. **小白（对手乐队吉他手）** - 长发男，对吉他抱有热情，被焱野缠上，最终两人在一起；技术很好，在主角学琴的时候提供过帮助，与后辈吉他手是劲敌关系（这个劲敌关系有点像宝可梦劲敌关系）

4. **live house店长（大哥系角色）** - 为爱发电，对音乐抱有热情，帮助主角熟悉设备与舞台知识，帮助主角在live house演出，对音乐节提供设备支持。

5. **拉拉菌（第一章主角乐队贝斯，吉祥物系角色）** - 为人温和，与三丰关系很好，但由于考学原因在第一章后离开乐队。

其他角色：

1. **乖酱/头孢（不定积分前任吉他手）** - 不喜欢说话的前吉他手，技术稳定但缺乏激情。选择在压力来临前主动退出，象征"理性止损者"。离开时没有背叛，只是选择了自我保护。

2. **千灯（外来的神秘吉他手）** - 传说中的技术高手，在社群中被神话。实际接触后发现只是另一个在音乐中寻找意义的普通人。象征"偶像的祛魅"。

3. **抽象学弟（社团边缘观察者）** - 始终在边缘徘徊，想加入但又害怕承诺。见证了整个崩溃过程，最终选择不加入任何阵营。象征"旁观者的无力"。

## 详细人物设定

### 1. 王行知（三丰）

**基本信息**

- **性别**：男
- **年龄**：18→24
- **性格关键词**：执念、责任感、技术洁癖、孤独、自省、光与热、灰烬、温柔

**外貌特征**  
平凡偏瘦，经典眼睛系阿宅；舞台灯下的专注让他看起来比现实更鲜活，摘了眼镜后眼神变化。

**动机与目标**  
用音乐证明自我价值，建立一个大家相互理解的团队，在不完美的世界中寻找纯粹。

**主要冲突**  
理想与现实的撕裂、信任与孤独的循环、自我燃烧与空虚的悖论。

**与主线关系**  
主线叙事的精神支点；所有人物与地标的命运都围绕他的执念旋转。

**成长弧**  
理想萌芽（学姐引路）→ 群体巅峰（第一次音乐节）→ 精神坍塌（动漫社演出）→ 重生（调音师之路）。

**象征体系**

- **象征意象**：信号塔——发出光的同时被风化。
- **象征物**：调音台与破损的琴弦。
- **主题色**：黑与深蓝（信念与倦怠的共存）。
- **音乐动机**：A 大调到 F#小调的循环——希望与疲惫的往复。

**人际关系**

- **关联角色**：包晏澜（学姐）、林知夏（贝斯手）、姜芮萌（后辈吉他手）、江昱（梓兮）、方芮宁（后辈女主唱）、赵启扬（鼓手）、马浩宇
- **关系类型**：羁绊、竞争、背叛、继承
- **互动描述**：他不断寻找'能理解他的人'，但每次都在误解中失去他们。

### 2. 包晏澜（包姐/学姐 - 精神引路人）

**基本信息**

- **性别**：女
- **年龄**：21～25
- **性格关键词**：成熟、疲惫、普通但有亲和力、理想主义者的终局、温柔、喜欢和稀泥

**外貌特征**  
长发，素食气质，笑中带怅然；像葛城美里式的忧郁偶像。

**动机与目标**  
在离开前把理想交托给后辈，让音乐延续。不想管太多事，但是又喜欢和稀泥。

**主要冲突**  
信仰失落与现实压力的撕扯。

**与主线关系**  
精神火种，点燃三丰的执念并成为永远的幻影。在主角心中，学姐是精神支柱，但是又无法真正理解主角。

**成长弧**  
和主角组乐队 → 前辈的荣光 → 排练小日常（小音箱往事/野鸡排练室，仔细三丰吵架日常调节） →新星剧场演出（唱的很好可靠）→ 考研转后勤 → 第一次音乐节 → 好感度剧情→ 化为回忆与信念。

**象征体系**

- **象征意象**：坠落的星（学姐的离开）
- **象征物**：破裂的耳返与旧 CD（学姐的离开）
- **主题色**：金与灰（光的残影）
- **音乐动机**：E 小调转 C 大调——消散的光辉与温柔的遗产

**人际关系**

- **关联角色**：三丰
- **关系类型**：导师与追随者
- **互动描述**：她是他的信仰起点，但永远无法被真正理解。

- **音乐动机**：B 小调主旋律，安静地支撑他人的和弦。

**人际关系**

- **关联角色**：王行知（三丰）、江昱（梓兮）
- **关系类型**：理解与错位
- **互动描述**：她懂得主角的痛，但无法阻止他坠落。

## 人物剧作画像

### 三丰

**叙事语气**  
低配旁白 + 自我诊断式独白；像在给万事万物事情发展做体检。

**核心隐痛**  
害怕'没有声音'的瞬间；沉默意味着自己白忙一场。

**行为逻辑**  
任何混乱里先抓'可控参数'（音量、延时、走位）；遇到质疑 → 加班校准 → 独自背锅。

**戏剧动能**  
把"理想=可调参数"的执念带进人际关系，直到关系也被当成旋钮拧坏。

**能力**  
能够通过调整参数来控制事情的发展，能够通过调整参数来控制人际关系的发展，能够通过调整参数来控制自己的情绪。

**场景触发**

- 设备出故障/舞台事故 → 进入"救场模式"并拒绝求助
- 被忽视或被篡改方案 → 失控式复盘长独白（含技术细节）

**声线与表演**  
独白轻、短、急促，像在赶着做事情；情绪上来时节奏加快但音量不升。

**视觉/镜头提示**

- CU：拨弦的细颗粒声，搬器材时的挥汗如雨
- OSD：叠加参数 HUD（-3dB、+12ms）随情绪抖动，设备故障时的报警声

**互动锚点**

- **对象：学姐** | 张力：信念遗产/误读之爱
- **对象：贝斯手** | 张力：稳定频率/现实温度
- **对象：梓兮** | 张力：镜像竞争/被取代恐惧
- **对象：后辈女主唱** | 张力：被表面之光吸引 → 被礼貌处决

**关键道具/动机音**  
调音台贴纸（旧校徽）、断了一根再缠回的琴弦；动机音：A→F#小调往复。

**示例台词**

- **独白**："别让这块场地沉下去……先把主混压到-3，再给人声开个微笑曲线，够了，够了……今天就让它活过这一首。"
- **对白**："你们看到的是灯，我看到的是能不能不爆炸。让我再试一次。"

**可选分支钩子**

- 承认'有人比我更会带队（其实并非有人）' → 人际和声开始出现（第三章）
- 继续一人控台 → 进入精神噪声段落（文本电波化）

### 学姐（精神引路人）

**叙事语气**  
像从舞台侧幕吹来的风；不解释，只带路。

**核心隐痛**  
知道'辉煌不可久居'，所以尽量把路照亮就退场。

**行为逻辑**  
把机会让给后辈；在崩溃前留下一套可执行范式（排练法/曲目栈）。

**戏剧动能**  
她的离开不是背叛是方法论的终点；逼迫三丰自我点火。

**场景触发**

- 闪回/天台短镜头 + 风声 + 不完整人声轨
- 毕业前夜把耳返递给主角：'别再对自己小声说话'

**声线与表演**  
轻到接近耳语；词句常留白。

**视觉/镜头提示**  
逆光半侧脸；镜头不过肩，避免正面定义她。

**示例台词**

- **对白**："别害怕空场。空场能让你听见自己。"

### 贝斯手（唯一救赎）

**叙事语气**  
把情绪翻译成节拍的工程师口吻；少言但句句在拍点上。

**核心隐痛**  
知道稳住别人很难稳住自己，犹豫时选择退出以免破坏频率。

**行为逻辑**  
先稳节奏再谈感情；冲突升级 → 暂停 → 下次更准时到场。

**戏剧动能**  
她的离开制造'低频缺失'，观众首次听见孤独的空洞；回归即 True 线信号。

**场景触发**

- 排练分歧 → 她摘耳机不争辩只换 click 值
- 第四/五章回归 → 先调监听再开口

**声线与表演**  
轻淡、中低；笑意更像呼吸。

**视觉/镜头提示**  
她在调监听时的眼神对位三丰；两人不用看对方也在同一拍点。

**示例台词**

- **对白**："你今天把低频让出来一点。我就回来了。"

**视觉/镜头提示**

## 系统用演出模块

### 共用镜头语法

- **文本电波化**：当主角精神噪声上行时，行距不齐、字重漂移、词中插入参数值（如 -3dB）
- **群像对话**：同屏多气泡，前景对白带轻微相位差，表现误解

### 选项设计范式

- **技术/关系二选一**：优先救声场 vs 先救人（影响后续信任线）
- **承认/否认**：承认他人方案优于己见 vs 坚持自我（决定是否进入第四章分歧）

## 旗标系统（Flags System）

### 主角核心旗标

| 旗标名            | 初始值 | 影响范围         | 触发阈值                     |
| ----------------- | ------ | ---------------- | ---------------------------- |
| `trust_kiki`      | 80     | 贝斯手关系线     | <50 触发离开；>70 触发回归   |
| `tech_obsession`  | 60     | 人际 vs 技术分歧 | >80 进入孤立线；<40 开启协作 |
| `ideal_faith`     | 100    | 主线方向         | <30 崩溃线；>70 坚持线       |
| `mirror_conflict` | 0      | 与梓兮关系       | >60 爆发对立；<20 和解可能   |
| `betrayal_count`  | 0      | 精神状态         | ≥3 进入噪声模式              |
| `music_fest_rep`  | 0      | 外界评价         | >100 声誉线；<-50 边缘化     |

### 关键选择与旗标变动

**第一章 - 学姐离开**
- 选项A："我会延续你的方式" → `ideal_faith +20`, `tech_obsession +10`
- 选项B："我会用自己的方式" → `ideal_faith +10`, `trust_kiki +15`

**第二章 - 第一次音乐节冲突**
- 优先调音台 → `tech_obsession +15`, `trust_kiki -10`, `music_fest_rep +20`
- 优先协调团队 → `tech_obsession -10`, `trust_kiki +20`, `music_fest_rep +10`

**第三章 - 梓兮顶替事件**
- 接受顶替 → `mirror_conflict +30`, `tech_obsession -15`, `betrayal_count +1`
- 拒绝让步 → `mirror_conflict +50`, `ideal_faith +10`, 团队分裂

**第四章 - 鸿门宴**
- 参与告别仪式 → `betrayal_count +1`, `ideal_faith -30`, 进入第五章重生线
- 拒绝参与 → `betrayal_count +2`, 孤立但保留尊严，进入调音师线

### 多结局旗标判定

**True End（贝斯手回归）**
- `trust_kiki ≥70`
- `tech_obsession <60`
- `ideal_faith ≥50`

**梓兮共沉线**
- `mirror_conflict ≥80`
- `betrayal_count ≥3`
- `ideal_faith <30`

**调音师独行线**
- `tech_obsession ≥80`
- `betrayal_count ≥2`
- `trust_kiki <40`

**功利成功线（马浩宇路径）**
- `music_fest_rep ≥120`
- `ideal_faith <40`
- 接受商业化选项 ≥3次

## 实施规范（基于 Prompt Pack）

### 输出约束

1. **严格遵循角色圣经**：任何对话/行为必须与本文档人设、矛盾、旗标状态一致
2. **禁止链式思考暴露**：不输出"因为他XXX所以XXX"，只输出行为与台词
3. **参数必须落地**：技术描述必须含具体数值（如-3dB、+12ms、432Hz）
4. **代价可视化**：每个选择必须标注旗标变动（格式：`[tech_obsession +15]`）
5. **双语关键词**：重要意象、台词标题提供中日对照
