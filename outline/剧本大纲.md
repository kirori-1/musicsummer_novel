# 剧本大纲

1. 宏观层（世界观 + 流程图式）

用途：确定整体结构、分支数量、游戏时长。

细节程度：像写论文提纲一样。只写

共通线：序章 → 日常 → 冲突 → 分歧点。

各分支：主冲突是什么，结局类型（True/Bad/Normal/Happy）。

字量：5000 字以内。

适用：提案阶段、Pitch、立项用。

## 第三章（2022 下半年–2023 初｜约 12–15 万字｜共通线 + 多 POV）

剧情基调（2022 ～ 2023 年）

- 精神污染与悬疑式，主角在高度压力的情况下思维逐渐模糊和偏执，思路变的电波，气氛急转直下
- 如何在所有人觉得一切都结束的时候强行安插新剧情呢，那就是完美的青春校园爽文转变为低气压党争番的剧情环节

  初步确定采用多线视角叙事，从个人英雄主义和宏大叙事转变为群像剧与利益纠葛感情纠纷
  在不同人物的视点会有不同的情绪风格和视角
  此章节可以多插入对不同热门梗的捏他部分，比如 mygo, 孤独摇滚，可以虚构很多乐队来当竞争对手，有发散空间
  功能：群像化，全面分裂，精神逐渐崩坏

起因：

主角考研失败 → 低谷归来，却发现大家各自忙碌，逐渐疏远。
