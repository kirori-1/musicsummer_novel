# 《音脉：学院都市 Chengdu Academia Loop》

## 一、世界观定位：现实成都的扩张性隐喻

> **一句话定义**  
> 在主城区霓虹与古迹之外，成都南部有一座被称为「学院都市」的巨大大学圈。  
> 数十所院校、社团、音乐工作室、青年之家构成一座“半自治的文化城邦”。  
> 它既是梦想温床，也是理想坟场。  
> 整个故事的青春、失真、理想与崩坏都在这片“学术环”中发生。

---

## 二、空间结构总览

| 区域层级 | 现实取材 | 功能与象征 | 叙事作用 |
|-----------|------------|-------------|------------|
| **核心圈：学院都市（University Loop）** | 天府新区、双流大学城、郫都大学区 | 数十所高校围成的“教育环城”，与成都主城区隔着一条“时差带”；城市节奏慢半拍 | 主角校园生活主舞台，乐队、社团、排练室、文化节都在此发生 |
| **连接带：空港廊道（Skyway Corridor）** | 双流机场、天府机场、成雅高铁 | “现实连接点”，学子进出城市的唯一通道 | 梦想与现实的界限：每一次离开成都，都要穿过这里 |
| **核心城：锦江文化区（Central Chengdu）** | 春熙路、九眼桥、太古里、东郊记忆 | 成都的“文化中枢”，真正的主流舞台与市场；但对学院都市的青年而言，它是遥远的神殿 | 乐队的目标地、精神终点、市场的象征 |
| **外围带：山地与古迹区** | 龙泉山、青羊宫、望江楼、文殊院 | 古代成都的精神延续与“心灵后门”；在城市精神层面上仍然影响年轻人的梦境 | 精神疗愈与顿悟的场所，崩溃与再生节点 |
| **地底层：地铁环线（Sub-Loop 7）** | 地铁7号线、19号线 | 命运回路的隐喻结构 | 不同章节的“转折通道”，每次坐地铁代表一次心境转变 |

## 三、学院都市的内部生态

| 模块 | 内容 |
|------|------|
| **社团生态** | 各高校有独立轻音部、动漫社、摄影社等，形成松散联盟（Music-Alliance）。内部竞争激烈，存在“校园流量战”。 |
| **音乐生态** | 学院都市内有几家半地下 Livehouse 和青年之家，设备简陋但热情浓烈。对外只能靠城市赛事和音乐节获得关注。 |
| **经济生态** | 由外包公司、赞助品牌、校园创业孵化中心控制——音乐节审批、赞助都与商业利益挂钩。理想与市场的碰撞集中在此。 |
| **精神结构** | 学生们生活在主城区的影子下，城市像个神明，既温柔又漠然。许多人终极目标是“去城里演一次”。 |

---

## 四、象征逻辑（地理＝心理）

| 地理象征 | 含义 | 对剧情的隐喻 |
|-----------|------|----------------|
| **机场** | “通往更大世界的出口”，但对大部分人只是望向的窗口 | 所有人都想飞走，却被困在入境口的生活现实里 |
| **地铁环线** | 命运与时间的循环 | 每次音乐节＝地铁一圈；序章与结尾可在同一站相遇 |
| **排练室** | 小宇宙、共振腔 | 群体关系的晴雨表——设备坏、被拆、重建象征理想脆弱 |
| **音乐节草坪** | 短暂的自由国度 | 每次音乐节都是“青春政权”的一次短暂建国 |
| **主城区霓虹** | 权力与资本的象征 | “主流”永远看似触手可及但遥不可攀 |

| 现实地点 | 设定转译 | 可用剧情功能 |
|-----------|------------|----------------|

| 成都地铁1号线、18号线 | 学院都市通往市区的“梦境通道” | 在地铁上切换时间与叙事 POV |
| 郫都大学城 | 学院都市北区 | 老排练室、低预算乐队起点 |
| 天府大学城 | 南区 | 新建社团总部、理想主义与管理体制的对撞 |
| 东郊记忆 | 主城区外缘文化带 | 乐队登上“大舞台”的试炼场 |
| 青年之家 | 市政支持的文化基地 | 理想与体制的边界；后期“疯魔化”核心舞台 |
| 春熙路–九眼桥–339塔 | 中心城市轴 | 梦想与幻灭的终点；鸿门宴发生点 |

## 六、叙事与空间的联动方式

| 剧章 | 空间焦点 | 叙事作用 |
|------|------------|------------|
| 序章（2016–2019） | 学院都市旧排练室 | 奠定“黄金年代”神话；为学院都市的建构提供历史根基 |
| 第一章（2020–2021） | 校园南区、空港廊道 | 疫情后复苏，青春热血的再启航 |
| 第二章（2021–2022） | 草地音乐节、青年之家 | 学院文化的“顶峰”；自组织文化盛宴 |
| 第三章（2022–2023） | 地铁环线、东郊记忆 | 精神裂解；理想被资本吞噬 |
| 第四章（2023） | 鸿门宴餐厅（主城区） | 理想被现实“宴请”；悲喜剧式的清算 |
| 第五章（后日谈） | 空港、火车站 | 现实出走与再出发；个体化职业轨迹 |

## 七、视觉与叙事语气建议

| 元素 | 风格建议 |
|------|-----------|
| **城市设计** | 校园区建筑色调偏冷灰+橘光（年轻气），主城区偏饱和暖色（消费热），机场与地铁用蓝白（逃逸感） |
| **音景（Soundscape）** | 学院都市：混响延迟、风声与远处鼓声；主城区：高频噪音、广播、电流；机场：压抑空洞感 |
| **叙述语气** | “成都不是故事的舞台，而是人物的情绪结构”——城市变化映射人物心态起伏。 |

## 八、后续开发方向

1. **学院都市结构图**  
   - 北区（旧厂文化）  
   - 南区（新建校区）  
   - 中环（青年之家）  
   - 外环（机场）

2. **角色空间分布表**  
   - 每个角色的出发点、心理空间与“归宿地”  
   - 例：三丰＝旧排练室→东郊记忆→地铁→机场

3. **叙事声景表**  
   - 每个章节的城市音色：风声、广播、交通、观众噪音

4. **机场象征线索**  
   - 飞机起降声作为章节转场或内心闪回的声学标志
