# 《学院都市 Chengdu Academia Loop》

## 一、世界观定位：现实成都的扩张性隐喻

> **一句话定义**  
> 在主城区霓虹与古迹之外，成都南部有一座被称为「学院都市」的巨大大学圈。  
> 数十所院校、社团、音乐工作室、青年之家构成一座“半自治的文化城邦”。  
> 它既是梦想温床，也是理想坟场。  
> 整个故事的青春、失真、理想与崩坏都在这片“学术环”中发生。

## 三、学院都市的内部生态

| 模块         | 内容                                                                                                     |
| ------------ | -------------------------------------------------------------------------------------------------------- |
| **社团生态** | 各高校有独立轻音部、动漫社、摄影社等，形成松散联盟（Music-Alliance）。内部竞争激烈，存在“校园流量战”。   |
| **音乐生态** | 学院都市内有几家半地下 Livehouse 和青年之家，设备简陋但热情浓烈。对外只能靠城市赛事和音乐节获得关注。    |
| **经济生态** | 由外包公司、赞助品牌、校园创业孵化中心控制——音乐节审批、赞助都与商业利益挂钩。理想与市场的碰撞集中在此。 |
| **精神结构** | 学生们生活在主城区的影子下，城市像个神明，既温柔又漠然。许多人终极目标是“去城里演一次”。                 |

## 五、时间与空间的现实融合点（含文化背景）

### 5.1 核心地标详解

#### 民族大学（学院都市主舞台）

**现实原型**: 西南民族大学（武侯校区）
**文化特征**:
- 多民族融合的校园文化，食堂有各族特色美食
- 老校区建筑带有80-90年代质感，橘黄路灯营造怀旧氛围
- 学校门口小吃街是学生社交中心，烧烤摊、奶茶店见证无数深夜谈心

**剧情功能**:
- 食堂排练室：前辈乐队的传说起点，设备简陋但回声效果意外好
- 校园操场草坪：第一次/第二次音乐节举办地，夕阳时分特别适合告别
- 新社团楼：第二代乐队的根据地，包晏澜在此建立"人人会调音"文化
- 天台：关键对话场景，可以看到远处城市的霓虹与飞机起降

**视觉设计要点**:
- 橘黄路灯+冷灰建筑的色调对比
- 食堂排练室的混响效果（声学特征）
- 天台远景可以看到339塔的灯光

---

#### 空港廊道（双流/天府机场）

**现实原型**: 双流国际机场T2 + 天府国际机场
**文化特征**:
- 成都南门的巨大交通枢纽，地铁10号线/18号线直达
- 机场候机楼的玻璃幕墙与飞机起降声营造"离别感"
- 深夜的机场大巴站，往往是最后的告别地

**剧情功能**:
- 包晏澜毕业离开的场景
- 第五章主角全国出差的起点
- 象征"逃离"与"追寻"的双重意义

**视觉设计要点**:
- 巨大的玻璃幕墙反射人影
- 飞机起降的轰鸣声作为情绪铺垫
- 地铁10号线的蓝白色调（冷色系）

**声景设计**:
- 广播提示音（440Hz标准音A）
- 行李箱滚轮声
- 远处的飞机引擎轰鸣

---

#### 地铁环线（命运回路）

**现实原型**: 成都地铁1/3/7/10号线
**文化特征**:
- 地铁7号线环绕主城区，是成都第一条环线
- 地铁1号线连接南北，从天府广场到韦家碾，穿越新旧成都
- 地铁站的瓷砖反光、报站女声、关门提示音构成独特声景

**剧情功能**:
- 时空转换的"魔法通道"
- 每次坐地铁代表人物心境转变
- 地铁报站声可以作为章节转场的声学标志

**关键站点**:
- **省体育馆站**（7号线）：靠近民大，学生常用站
- **火车南站**（1/7号线交汇）：离别与归来的中转站
- **太平园站**（3/7号线交汇）：去东郊记忆的换乘点
- **春熙路站**（2/3号线交汇）：主城区中心，鸿门宴前的地铁站

**视觉设计要点**:
- 车窗反射人物面容（镜像效果）
- 地铁隧道的闪烁灯光
- 站台等车时的疏离感

**声景设计**:
- 报站女声："下一站，春熙路。Next station, Chunxi Road."
- 关门提示音（蜂鸣声）
- 隧道中的轰鸣与车厢内的寂静对比

---

#### 青年之家（理想与体制的边界）

**现实原型**: 各区青少年宫/青年之家活动中心
**文化特征**:
- 政府支持的青年文化空间，有排练室、多功能厅
- 需要申请审批，象征"体制内的自由"
- 设备比学校好，但管理更严格

**剧情功能**:
- 第二章：方芮宁开始主导集合排练的场所
- 第四章：三丰孤军筹备第二次音乐节的"独角戏"现场
- 大雨舞台怒骂的发生地

**视觉设计要点**:
- 多功能厅的白炽灯（冷白色调）
- 标准化的舞台设备vs破旧的学校排练室
- 空旷的走廊与回声

---

#### 东郊记忆（主城区外缘文化带）

**现实原型**: 东郊记忆·成都国际时尚产业园
**文化特征**:
- 原红光电子管厂改造，工业遗址风格
- 成都indie音乐/live house的聚集地
- 砖红色厂房+涂鸦墙+铁轨，充满后工业美学

**剧情功能**:
- 对手乐队（马浩宇）的主场
- "大舞台"的象征，比学校舞台更专业但也更商业化
- 可能的川大演出/对外合作演出场地

**视觉设计要点**:
- 砖红色厂房外墙
- 铁轨与涂鸦墙
- livehouse昏暗的舞台灯光

---

#### 春熙路商圈（主城区中心/鸿门宴）

**现实原型**: 春熙路步行街 + 太古里 + IFS
**文化特征**:
- 成都最繁华商圈，象征"主流"与"成功"
- 太古里的精致咖啡馆vs春熙路的接地气小吃
- IFS的爬墙熊猫是成都网红地标

**剧情功能**:
- 鸿门宴餐厅的选址（高档餐厅，象征"体面的告别"）
- 与学院都市形成强烈对比：学生区的破旧vs市区的繁华
- 方芮宁选择这里举办告别宴，强化"温柔暴力"的意味

**餐厅选址建议**:
- 太古里某高档川菜餐厅（需要预约，有包间）
- 或IFS附近的融合料理餐厅
- 落地窗可以看到春熙路的霓虹灯

**视觉设计要点**:
- 餐厅包间的暖黄灯光
- 落地窗外的繁华街景（模糊处理）
- 精致的餐具vs内心的荒凉

---

#### 新星剧场（校园文化的高光舞台）

**现实原型**: 校内大礼堂/剧场
**文化特征**:
- 学校最正式的演出场地，有专业灯光音响
- 对学生而言是"梦想舞台"
- 需要申请，象征"官方认可"

**剧情功能**:
- 前辈乐队的高光演出（包晏澜主唱时期）
- 第二代乐队的重要演出节点
- 方芮宁与江昱打造"新正统"的舞台

**关键事件**:
- 2019：前辈乐队演出（包晏澜主唱）
- 2021春夏：第二代出道演出
- 2023春：方芮宁+江昱的"新正统"演出

---

#### CH8调音室（第五章核心场景）

**现实原型**: 成都本地音乐工作室/livehouse后台
**文化特征**:
- 专业的调音设备，职业化的工作环境
- 从"学生玩票"到"职业调音师"的转变
- 象征理想被"格式化"但也被"保存"

**剧情功能**:
- 第五章主角入职后的工作场所
- 学习专业调音技术的地方
- 全国出差的起点

**视觉设计要点**:
- 调音台的荧光屏
- 专业监听音箱
- 工作台上的咖啡与乐谱

## 七、视觉与叙事语气建议

| 元素                   | 风格建议                                                                                    |
| ---------------------- | ------------------------------------------------------------------------------------------- |
| **城市设计**           | 校园区建筑色调偏冷灰+橘光（年轻气），主城区偏饱和暖色（消费热），机场与地铁用蓝白（逃逸感） |

## 八、视觉小说制作的空间设计指南

### 8.1 场景复用策略（降低制作成本）

**核心原则**: 同一空间不同时间/光线/人物组合=不同场景

#### 高频场景（需要多套CG）

1. **天台**（至少3套CG）
   - 白天版：阳光明媚，用于日常对话
   - 黄昏版：夕阳西下，用于告别/深度对话
   - 夜晚版：城市灯光背景，用于秘密谈心

2. **排练室**（至少4套CG）
   - 旧排练室（食堂）：前辈时代，暖色调
   - 小音箱排练室：第一代，设备简陋
   - 新社团楼排练室：第二代，设备改善
   - 青年之家排练室：标准化，冷色调

3. **地铁站/车厢**（至少2套CG）
   - 车厢内部：窗户反射人影
   - 站台：等车的孤独感

4. **校园操场草坪**（至少3套CG）
   - 白天排练
   - 夕阳音乐节
   - 夜晚收场

#### 中频场景（1-2套CG）

- 学校门口小吃街
- 机场候机厅
- 鸿门宴餐厅
- 川大演出场地
- 动漫社剧场

#### 低频场景（可用文字+音效）

- 出租屋（江昱/主角）
- livehouse后台
- CH8调音室
- 火车站

---

### 8.2 场景切换的逻辑规则

#### 时空跳跃的过渡方式

1. **地铁转场**
   ```
   [场景A] → 地铁报站声 → 黑屏/车窗倒影 → [场景B]
   适用：跨区域移动（学校→市区）
   ```

2. **声音桥接**
   ```
   [排练室吉他声] → 渐弱 → [天台风声] → 渐强
   适用：同一空间不同区域
   ```

3. **时间淡入淡出**
   ```
   [夕阳版CG] → 淡出 → 字幕"三个月后" → 淡入 → [夜晚版CG]
   适用：时间跨度较大
   ```

4. **人物POV切换**
   ```
   [三丰视角-排练室] → 黑屏+独白 → [江昱视角-出租屋]
   适用：第三章多POV叙事
   ```

### 8.3 声景设计总表（Sound Design）

| 空间类型   | 主要音效                   | 次要音效               | 氛围音乐                 | 情绪标签           |
| ---------- | -------------------------- | ---------------------- | ------------------------ | ------------------ |
| 排练室     | 乐器声、设备故障音         | 脚步声、门开关声       | lo-fi hip hop（练习时）  | 专注、疲惫、冲突   |
| 天台       | 风声、远处交通声           | 飞机起降声、鸟鸣       | 环境音主导，极少音乐     | 孤独、告别、顿悟   |

### 8.7 音效与BGM的触发逻辑

#### 关键道具的声音标识

- **包姐的耳返**：底噪50Hz电流声 → 每次出现自动播放
- **调音台**：按钮点击音、推子滑动音 → 强化"技术洁癖"
- **破损琴弦**：断弦的"嘣"声 → 象征信任断裂

#### 情绪BGM触发规则

```
[剧情张力值] 决定BGM强度
- 张力 <30：环境音为主，极少音乐
- 张力 30-60：轻柔的lo-fi或钢琴曲
- 张力 60-80：情绪化的弦乐/电子乐
- 张力 >80：高潮演出曲目/压抑的噪音音乐
```

#### 场景专属BGM示例

- **天台系列**：风声+远处城市白噪音+偶尔的飞机声
- **排练室系列**：设备底噪+乐器调音+成员对话
- **地铁系列**：轰鸣声+报站声+车厢内的寂静（强烈对比）
- **音乐节**：现场演奏录音+观众欢呼+设备故障音

## 九、世界观的显示条件设计

### 9.1 渐进式世界观揭示

**原则**: 不一次性倾倒世界观，而是随剧情推进逐步扩展玩家的认知地图

#### 序章（2016-2019）
**可见空间**: 民大校内（食堂、旧排练室、天台）
**视觉风格**: 怀旧滤镜，暖色调
**玩家认知**: "这是一个关于学校乐队的故事"

#### 第一章（2020-2021上）
**新增空间**: 新社团楼、小音箱排练室、校园操场
**视觉风格**: 明亮自然光，充满希望
**玩家认知**: "原来学院都市不止一个学校，还有更多排练室"

#### 第二章（2021下-2022上）
**新增空间**: 青年之家、草地音乐会、新星剧场、地铁站
**视觉风格**: 暖黄+粉橙，夕阳质感
**玩家认知**: "学院都市与主城区之间有地铁连接，开始感受到'更大世界'的存在"

#### 第三章（2022秋-2023春）
**新增空间**: 地铁环线（多个站点）、川大、动漫社剧场、出租屋
**视觉风格**: 冷灰+蓝紫，阴影加重
**玩家认知**: "原来主角的活动范围一直局限在学院都市，现在开始跨校、跨区"

#### 第四章（2023夏）
**新增空间**: 春熙路商圈、鸿门宴餐厅、机场廊道
**视觉风格**: 暗红+黑灰，舞台强光与深色阴影
**玩家认知**: "终于到达'主城区'，但这是以背叛和告别为代价的"

#### 第五章（2023秋-2024）
**新增空间**: CH8调音室、其他城市（文字描述）、livehouse网络
**视觉风格**: 蓝灰+木色，职业化
**玩家认知**: "世界观完全打开，主角不再局限于成都，音乐成为职业"

### 9.4 镜头语言与演出指引

#### 常用镜头类型

| 镜头类型   | 使用场景                 | 情绪效果           | 实现方式                   |
| ---------- | ------------------------ | ------------------ | -------------------------- |
| 远景       | 音乐节全景、城市天际线   | 宏大、孤独         | CG绘制，俯视或平视角度     |
| 全景       | 排练室全貌、舞台演出     | 群像、氛围         | 包含多个角色的CG           |
| 中景       | 日常对话、排练场景       | 自然、关系         | 立绘+背景                  |
| 近景       | 关键对话、情绪时刻       | 亲密、紧张         | 立绘特写+表情差分          |
| 特写       | 耳返交接、手部动作       | 象征、细节         | 局部CG（道具/手/眼睛）     |
| 主观镜头   | 主角视角的舞台、调音台   | 代入感             | POV-CG，第一人称视角       |
| 反射镜头   | 地铁车窗、镜子、玻璃幕墙 | 镜像、自我审视     | 特殊CG，人物叠加反射效果   |

#### 特殊演出效果

1. **文本电波化**（第三/四章）
   - 行距不齐、字重漂移
   - 插入技术参数（-3dB、432Hz）
   - 闪烁效果

2. **多气泡对话**（群像冲突）
   - 同屏3-4个对话框
   - 轻微相位差，表现误解
   - 字体颜色区分角色

3. **画面失真**（精神崩溃）
   - 色彩饱和度降低或过高
   - 画面抖动、撕裂
   - 噪点增加

4. **慢镜头定格**（关键时刻）
   - 耳返交接瞬间
   - 鸿门宴举杯定格
   - 第二次音乐节怒骂后的沉默
